execute

Integer execute(InputStream input, OutputStream out, OutputStream err, Environment env)
Content copied to clipboard

Executes the command (see Command.execute).

If a filename is given, reads its contents as input. Else reads from input instead of file.

Finds the matches of the pattern in input. Then prints the lines containing the matches to out delimited by \n.

If "-A" flag is provided:

  • overlapping and consecutive regions are merged

  • when printing, regions are delimited by --\n

Does not support huge files (> 2 GB).

Does not support multiline patterns.

Return

  • 0 on success

  • 1 if argument list is invalid

  • 2 if an java.io.IOException happens during reading file

  • 3 if pattern is invalid

Parameters

input

Stream to read input from in case no filename is given.

out

Stream to print output to.

err

Stream to print errors to.

env

Ignored.

Throws

ru.hse.ezh.exceptions.ExecutionIOException

If input or out stream error occurred.
